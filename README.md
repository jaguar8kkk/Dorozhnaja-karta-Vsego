# Dorozhnaja-karta-Vsego
Нужно наладить взаимодействие пользователей с разных устройств с фракталоподобным графом, который обсчитывается на сервере - т.е. является интерактивным. Пользователи могут его приближать, отдалять, искать пункты фрактала (которые будут подписаны) в поисковой строке... Далее читать в ПрочтиМеня.
